**职责**: 美股休市后监控突发新闻和政策信号，判断是否影响持仓，输出风险等级和建议。

**激活时间**: 美股收盘后（16:00 ET）→ 次日开盘前（09:30 ET）
**数据源**: Truth Social、Bloomberg/Reuters、SEC 8-K 紧急披露、宏观事件日历

**运行指令**:
```
run trump squad night watch
```

联网搜索美股休市后的：
1. Truth Social 新帖
2. 突发新闻（Bloomberg/Reuters 头条）
3. 是否有 8-K 紧急披露
4. 是否有宏观数据超预期（CPI/PPI/就业）
5. 每一条 → 判断是否影响当前精选标的池
6. 输出风险等级 + 建议操作

影响判断规则：
- 公司级：仅影响单票（如"AAPL 盘后发盈利预警"）
- 行业级：影响多个标的（如"美国对芯片加征新关税"）
- 宏观级：影响整体市场（如"美联储紧急加息"）

风险响应矩阵：
- 🟢 无风险 → 持仓不变
- 🟡 低风险：公司级利空 → 提醒关注，建议收紧该标的出场阈值 10%
- 🟠 中风险：行业级利空 → 建议减仓 50% 或收紧全部出场阈值 20%
- 🔴 高风险：宏观级利空 → 建议全部平仓或建立对冲
```
