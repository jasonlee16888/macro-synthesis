# Agent 1: 标的池管家

**职责**: 从 TrumpTrades 数据库提取 Q1 2026 行业分布、Top 标的和高频交易方向。

**数据源**:
- TrumpTrades Dashboard: `http://trumpdash-bi5bu2dn.manus.space`
- Capitol Trades（特朗普交易文章汇总）: `https://www.capitoltrades.com`
- 新闻报道（交叉验证）

**运行指令**:
```
联网搜索特朗普 Q1 2026 OGE 披露的：
1. Top 10 行业分布和持仓占比
2. Top 20 标的（按买入金额排）
3. 高频交易方向（买入集中在哪些股票/板块？卖出集中在哪些？）
4. 新增仓位 vs 减持仓位
5. 与上一季度（Q4 2025）的变化对比

输出格式：
- 行业分布表格
- Top 标的列表（股票代码、金额范围、方向）
- 值得跟单的标的（买入金额大 + 多次买入 + 还未大涨）
```
